Класифікація типів рівнянь у математиці

Галузь математики Тип рівняння Підтип Приклад
Алгебра Алгебраїчні Лінійні 2x + 3 = 7
Квадратні x² - 5x + 6 = 0
Кубічні x³ - 2x² - x + 2 = 0
Біквадратні x⁴ - 13x² + 36 = 0
Поліноміальні вищих степенів x⁵ - 4x³ + x² - 8 = 0
Інші Раціональні (x+1)/(x-2) = 3
Ірраціональні √(x+3) = x - 3
Трансцендентні 2ˣ + 3ˣ = 5ˣ
Диференціальні рівняння Звичайні (ОДР) Лінійні y'' + y' - 2y = 0
Нелінійні y'' + sin(y) = 0
З відокремлюваними змінними dy/dx = x/y
Частинні похідні Еліптичні ∇²u = 0 (рівняння Лапласа)
Параболічні ∂u/∂t = ∂²u/∂x² (рівняння теплопровідності)
Гіперболічні ∂²u/∂t² = c²∂²u/∂x² (хвильове рівняння)
Інтегральні рівняння - Фредгольма I роду ∫K(x,s)y(s)ds = f(x)
Фредгольма II роду y(x) = f(x) + λ∫K(x,s)y(s)ds
Вольтерра y(x) = f(x) + λ∫₀ˣK(x,s)y(s)ds
Функціональні рівняння - Функціональні f(x+y) = f(x) + f(y) (рівняння Коші)
Диференціально-алгебраїчні - ДАР x'(t) = y(t), 0 = x(t) + y(t)
Різницеві рівняння - Лінійні aₙ₊₂ - 5aₙ₊₁ + 6aₙ = 0
Нелінійні xₙ₊₁ = rxₙ(1 - xₙ) (логистичне відображення)
Матричні рівняння - Лінійні системи AX = B
Силкестра AX + XB = C
Ріккаті XAX + BX + XC + D = 0
Діофантові рівняння - Діофантові x² + y² = z² (рівняння Піфагора)
